# CryptoSafe
Простой, надежный и локальный менеджер паролей с открытым исходным кодом, написанный на Python. Программа позволяет безопасно хранить учетные данные, шифруя базу данных на стороне пользователя.

========================================================================

ОПИСАНИЕ ПРОЕКТА
----------------
Данное программное обеспечение представляет собой легковесный и полностью 
автономный менеджер для безопасного хранения и управления личными 
паролями, логинами, адресами сайтов и другими текстовыми данными.

Приложение работает на базе концепции "Zero-Knowledge" (Нулевое разглашение) 
в режиме 100% оффлайн. База данных хранится локально на устройстве 
пользователя, что исключает риски перехвата данных в сети или утечек 
с удаленных облачных серверов.

КЛЮЧЕВЫЕ ОСОБЕННОСТИ И БЕЗОПАСНОСТЬ
----------------------------------
1. Криптографическое ядро: Данные шифруются "на лету" по стандарту 
   AES-128 (технология Fernet) в режиме CBC с аутентификацией HMAC.
2. Защита от перебора (Брутфорса): Мастер-пароль не хранится в базе. 
   Криптографический ключ генерируется в ОЗУ с помощью функции PBKDF2 
   и 400 000 итераций хеширования SHA-256 со случайной солью.
3. Защита от SQL-инъекций: Все манипуляции с локальной базой данных SQLite3 
   происходят через подготовленные параметризованные запросы.
4. Постквантовая устойчивость: Связка алгоритмов AES и SHA-256 признана 
   математически устойчивой к вычислениям на квантовых компьютерах.
5. Интеллектуальный генератор: Создание взломостойких паролей (длиной от 
   10 до 64 символов) на базе модуля "secrets" (энтропия ядра ОС).

АРХИТЕКТУРА И МОДУЛЬНАЯ СТРУКТУРА
---------------------------------
Проект разработан в соответствии с принципами разделения обязанностей (SoC):
- crypto_engine.py  -- Математика шифрования и генератор паролей.
- db_manager.py     -- Прямое безопасное взаимодействие с СУБД SQLite3.
- gui_components.py -- Графическая HUD-оболочка (PyQt6) и логика кэширования.
- main.py           -- Точка входа, координирующая запуск системы.

ПРОИЗВОДИТЕЛЬНОСТЬ И UX
----------------------
- Поиск на лету: Фильтрация базы осуществляется мгновенно прямо в 
  оперативной памяти через пакетный кэш `self.all_records`.
- Константная оптимизация O(1): Метод переключения видимости паролей 
  `self.target_row_idx` управляет элементами точечно, не нагружая CPU 
  линейными циклами перебора таблицы.
- Приватность: Реализован автоматический скрытый режим просмотра (пароли 
  под звездочками), быстрая отправка в буфер обмена и динамическая смена 
  дизайна (Тёмная, Светлая, Серая темы) с сохранением настроек.

ИНСТРУКЦИЯ ПО ЗАПУСКУ ИЗ ИСХОДНОГО КОДА
---------------------------------------
1. Установите зависимости проекта:
   pip install -r requirements.txt

2. Запустите главный файл приложения:
   python main.py

СБОРКА АВТОНОМНОГО .EXE ФАЙЛА
-----------------------------
Для компиляции проекта в единый исполняемый файл для Windows используется 
утилита PyInstaller. Команда для сборки без фонового окна консоли:

pyinstaller --onefile --noconsole main.py

После завершения сборки готовый файл будет находиться в папке /dist.
